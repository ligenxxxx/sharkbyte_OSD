# sharkbyte_OSD

## bmp format
    24bits color.

## size(width x height): 
    large: 42x42  
    medium: 24x36  
    small: 12x18  

## bmp file naming convention: 
    username + '_' + index + '.bmp'  eg: ligenxxxx_000.bmp
    everyone can design the osd bmp file, and PR to this repositories.
    each designer has own index starting from 0.
